cargarOraciones(archivo_entrada, CargarInicioFinTokens=True)

source code 

Retorna la lista de oraciones, cargadas desde un archivo de texto.

Cada elemento es de tipo [(dic, tag_evento)] donde dic tiene claves 'token', 'lema' y 'tag_freeling' (eventualmente más)

CargarInicioFinTokens determina si se espera que se carguen las posiciones de inicio y fin de cada token, o si esta información no está en el archivo.

printOraciones(oraciones, archivo, ImprimirInicioFinTokens=True)

source code 

Imprime oraciones a un archivo de texto.

En ImprimirInicioFinTokens se indica si se quiere imprimir la información de inicio y fin de los tokens.

printOracionesSalidaClasificacion(oraciones, archivo)

source code 

Imprime oraciones a un archivo de texto.

En cada oración el diccionario con los atributos debe tener la clave tag_clasificador.
